CarCare Companion

A Python-based vehicle maintenance assistant that helps car owners with problem diagnosis, expense tracking, and mileage optimization.

About

CarCare Companion is a college project . It provides an integrated solution for vehicle maintenance management through a simple menu-driven interface.
